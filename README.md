## About This Portfolio

This portfolio showcases my transition from 10 years of marketing management and logistics operations into AI training and data quality roles. It highlights:

- **AI Competencies** — Data annotation, prompt engineering, RLHF, content quality review
- **Professional Background** — Marketing management, data analysis, Excel skills, quality assurance
- **Continuous Learning** — AI certifications, online courses, hands-on AI projects
- **Remote Readiness** — Available for international remote AI training opportunities

## Features

- **Responsive Design** — Works on mobile, tablet, and desktop
- **Dark Mode** — System preference detection with localStorage persistence
- **WCAG 2.1 AA Compliant** — Accessible to screen readers and keyboard users
- **Contact Form** — Client-side validated form with live error feedback
- **SEO Optimized** — Open Graph, Twitter Card, semantic HTML
- **Printable Resume** — Clean resume page for `Cmd/Ctrl + P`
- **Availability Badge** — Green "Available for Remote AI Roles" indicator
- **Zero Dependencies** — Pure HTML, CSS, and JavaScript

## Customization

**Update personal info** in `index.html`:
- Name, bio, and AI experience details
- Contact email and LinkedIn URL
- Certifications and course names in resume

**Update colors** in `style.css`:
```css
:root {
  --primary: #2563eb;     /* Main blue */
  --accent: #0891b2;      /* Teal accent */
}
```
